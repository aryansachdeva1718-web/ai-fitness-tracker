# Recovery Feedback System Ideas

Future feature.

After calculating recovery score:

Ask user for optional recovery feedback.

Example:

How did recovery feel today?

- Great
- Fine
- Poor
- Very Fatigued

Or allow free text notes.

Example:

"Felt weaker on bench today"

"Legs still sore"

"Low energy during session"

---

Purpose

Compare system prediction against actual user experience.

Example

Predicted Recovery = 85

User Feedback = Poor

This may indicate scoring system needs improvement.
